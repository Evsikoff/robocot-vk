# Статические URL для изображений

Загрузите статические изображения в корень раздачи так, чтобы они были доступны по следующим URL (относительно домена приложения, например `https://robocot.vercel.app`):

- `/apple-touch-icon.png` — иконка Apple Touch 180×180 для iOS.
- `/favicon-32x32.png` — фавиконка 32×32 для десктопа.
- `/favicon-16x16.png` — фавиконка 16×16.
- `/safari-pinned-tab.svg` — маска для pinned-tab Safari.
- `/site.webmanifest` — манифест приложения с описанием иконок.
- `/icon-192x192.png` — иконка PWA 192×192 из манифеста.
- `/icon-512x512.png` — иконка PWA 512×512 из манифеста.
- `/fb.png` — картинка превью для Open Graph (используется в `<meta property="og:image">`).

Убедитесь, что пути совпадают точно, чтобы браузер и социальные сети могли корректно загрузить ресурсы без 404.
